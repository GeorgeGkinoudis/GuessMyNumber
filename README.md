- # Guess My Number <br>

This is a small JavaScript project I built while learning JavaScript through a Udemy course. The goal was to practice the fundamentals by creating a simple interactive browser game. <br>

- ## About the Project<br>

**Guess My Number** is a number-guessing game where the player tries to guess a randomly generated number between 1 and 20. The game gives feedback if the guess is too high or too low, keeps track of the score, and updates the high score. <br>

- ## What I Used<br>

* JavaScript fundamentals (variables, conditionals, functions)<br>
* DOM manipulation<br>
* Event listeners<br>
* Random number generation with `Math.random()`<br>
* Updating HTML content dynamically<br>
* Changing CSS styles with JavaScript<br>
* Score and highscore logic<br>

- ## Project Files

index.html – structure<br>
style.css – styling<br>
script.js – game logic<br>
